**To explain [test doubles in PHPUnit] by using various examples.**

## Usage

First, run the following command to install/update Composer packages:

```bash
composer update -n
```

Now we can run PHPUnit tests:

```bash
./vendor/bin/phpunit
# or
./vendor/bin/phpunit --debug
```
